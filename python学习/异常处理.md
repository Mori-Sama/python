# 异常处理

异常处理的三个特征：追踪信息，类型，内容

~~~python
try:
    # 可能抛出异常的代码
    # 从被检测到异常代码开始，之后的代码将不再执行
    # try不能单独与else连用，finally可以与try单独使用
except 异常类型 as e:
    # 异常类型可以为一个元组
    # 万能异常Exception，它是所有异常的父类
    pass
except 异常类型2 as e:
    pass
else:
    # 如果被检测的代码块没有异常发生，则执行else代码
finally:
    # 无论被检测的代码块有没有异常，都会执行
    # 一般写的是回收系统资源的代码

# 能用if判断解决的错误，就不要用try
~~~
